# 第二编 设计工程

> 第6-26章 | 航空航天阀门设计全流程工程实践

---

## 本编概述

第二编是Avis知识体系的核心设计工程篇，覆盖从驱动方式选择、结构力学分析、密封工程设计到各类阀门（电磁阀、气动阀、液压阀、电机驱动阀、火工阀、单向阀等）的详细设计方法，同时涵盖精密加工、表面工程、增材制造等先进制造技术。本编共21章，是从事阀门设计工程师的必读内容。

## 学习路径

- **设计工程师路径**：第6章(驱动选型) → 第7-9章(力学基础) → 第10章(需求开发) → 按需深入具体阀门类型章节
- **制造工程师路径**：第18章(精密加工) → 第19章(表面工程) → 第22章(装配) → 第23章(无损检测) → 第24章(增材制造)
- **系统工程师路径**：第10章(系统工程) → 跨越式浏览各类阀门设计章节 → 重点研读第25-26章(系统级应用)
