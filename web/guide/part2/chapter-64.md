## 64.1 飞机燃油系统架构与阀门功能定位

### 64.1.1 飞机燃油系统的阀门全景

飞机燃油系统（Aircraft Fuel System）从油箱到发动机燃油喷嘴的整个流体链中，阀门是实现燃油存储、输送、分配、应急和安全管理的基础元件。现代大型客机燃油系统包含超过30个各类阀门，军用战斗机因空中加油和副油箱抛放等需求，阀门配置更为复杂。

燃油系统阀门按功能定位分五大类：

| 功能类别 | 典型阀门 | 关键性能要求 |
|---------|---------|------------|
| 增压输送 | 燃油增压泵、引射泵被动阀、供油关断阀 | 高流量(MJ级)、低压(<50psi)、长寿命 |
| 分配交输 | 交输阀(Crossfeed)、油箱选择阀 | 全开/全关、差压小(<5psi) |
| 空中加油 | 受油阀、加油关断阀、通气阀 | 快速对接密封、大流量 |
| 惰化安全 | NGS分配阀、氧传感器反馈阀 | 低泄漏、防爆认证 |
| 应急安全 | 应急放油阀(Jettison)、防火关断阀 | 快速全开、防火1100°C/15min |

### 64.1.2 大型客机燃油系统架构（以波音787和空客A350为参考）

**波音787燃油系统**采用全电设计（More Electric Aircraft理念）：

- 三油箱布局：左主油箱、右主油箱、中央油箱，总容量约126,000L
- 电动燃油泵（变频电机驱动，取代传统液压马达泵）
- 电动交输阀、电动加油/放油阀，取消液压作动
- 氮气惰化系统(NGS)覆盖全部燃油箱
- 阀门由通用数据网络(GDN)和远程数据集中器(RDC)控制

**空客A350燃油系统**：

- 总容量约156,000L（A350-1000），三油箱加配平油箱(Trim Tank)
- 部分保留液压作动阀，逐步向电动过渡
- 燃油管理计算机(FCMC)集成控制全部阀门
- 重心控制(CG Control)通过配平油箱前后传输阀实现

### 64.1.3 战斗机燃油系统架构（以F-35和F-22为参考）

**F-35闪电II燃油系统**：

- 内部燃油约8,300kg（F-35A），分布在机翼整体油箱和机身油箱
- 燃油同时作为热沉（PTMS：动力与热管理系统），燃油系统阀门必须额外耐受高温燃油（>150°C返回油箱）
- 空中加油受油阀兼容硬管（USAF）和软管-锥套（USN/USMC）双模式
- 副油箱抛放阀含爆炸螺栓作动的快速脱落阀
- 全部阀门由VMS（飞机管理系统）通过1553B数据总线控制

**F-22猛禽燃油系统**：

- 内部燃油约8,200kg，独有超音速巡航(>M1.5)时的燃油管理能力
- 燃油系统阀门全部按MIL-DTL-83833标准认证
- 防冰/防微生物处理阀

### 64.1.4 直升机与无人机燃油系统特点

**直升机**：燃油箱靠近重心，管路短，阀门以关断阀和交输阀为主。重要特征：抗坠毁燃油关断阀（Crash-Resistant Fuel Shutoff Valve），在冲击载荷>20g时自动切断所有油路，满足MIL-DTL-27422抗坠毁燃油系统标准。

**无人机（UAV）**：高空长航时(HALE)无人机（如Global Hawk）燃油系统突出高空低压环境（>60000ft）阀门防气蚀设计；高机动无人作战飞机(如XQ-58A)类似战斗机要求。

## 64.2 燃油增压与供输油阀

### 64.2.1 燃油增压泵与关断阀

**燃油增压泵（Boost Pump）出口关断阀**：

每台增压泵出口设置一个电动作动关断阀（Motor-Operated Valve, MOV）或电磁关断阀（Solenoid Shutoff Valve）。核心功能：
- 泵关闭时自动关闭，防止倒流
- 泵故障时隔离，由交输阀从其他油箱供油
- 维护时可手动锁定在关闭位

典型设计参数：

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 额定流量 | 15,000-35,000 pph（约100-240 L/min） | 对应单台发动机燃油消耗 |
| 工作压力 | 15-50 psi（增压泵出口） | 保证发动机FMU进口正压 |
| 作动方式 | 28VDC电磁阀或115VAC电动 | MIL-STD-704电源兼容 |
| 响应时间 | 全关<1s，全开<3s | 故障隔离要求 |
| 泄漏量 | 内漏<5 cc/min @ 1.5倍额定压力 | 防止油箱间燃油转移 |
| 寿命 | ≥50000次循环 | 对应>80000飞行小时 |

**阀门结构**：球阀或蝶阀为主，阀座采用PTFE或PEEK复合材料，与Jet A-1兼容（体积溶胀<1%，按ASTM D471）。电动执行器含手动超控手柄，机组可通过地板面板手动操作。

### 64.2.2 交输供油阀（Crossfeed Valve）

**功能与设计**：

交输供油阀连接左右侧供油管路，实现：
1. **发动机交输供油**：一侧油箱向另一侧发动机供油，应对泵故障或非对称耗油
2. **APU供油**：通常从左侧供油管路经交输阀至APU
3. **地面加油分配**

**设计要点**：
- 全开/全关双位阀，无中间位置需求
- 差压极小（<5psi），对流通能力要求高（Cv>50）
- 球阀为主流选择，大口径（通常2-3英寸）
- 电动执行器，含位置指示微动开关，向EICAS/ECAM提供阀门位置反馈
- 失电保持当前位置（fail-as-is）或弹簧返回关闭（spring-return close）

**双交输阀配置**：部分双通道客机（如B777）采用双重交输阀——前交输和后交输，提供冗余和隔离灵活性。

### 64.2.3 燃油引射泵（Jet Pump）被动阀

引射泵（也称喷射泵、Eductor Jet Pump）是燃油箱内无活动部件的"被动泵"，利用增压泵的高压引射流（Motive Flow）卷吸周围燃油。没有传统阀门，但其**被动止回阀（Check Valve）**功能值得分析：

- **无回流设计**：引射泵喉部后设置扩散段和止回挡板，当引射流停止时防止倒流
- **被动阀失效模式**：挡板弹簧疲劳是唯一失效模式，相较主动阀可靠性高一个数量级
- **油箱底部集油**：引射泵将油箱底部积水/杂质连续输送至发动机消耗，防止微生物滋生（替代传统放沉淀阀）

## 64.3 空中加油阀门

### 64.3.1 硬管加油（Boom）受油阀

**系统概述**：美国空军标准空中加油方式。加油机（如KC-135、KC-46）伸出伸缩套管（Boom），对接受油机背部的**受油插座（Receptacle）**。受油阀位于插座后部，由机械触发开启。

**受油阀设计**：

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 额定加油流量 | 1000-1200 USGPM（约3800-4500 L/min） | 硬管加油高流量优势 |
| 开启机制 | 机械触发（Boom插入力）+ 先导压力 | 双重触发确保对接后才开启 |
| 密封设计 | 双道O形圈+金属对金属锥面密封 | 快速对接/脱离零泄漏 |
| 脱离机制 | 弹簧复位 + 燃油压力辅助关闭 | 应急脱离<0.5s关闭 |
| 耐压 | 100 psi（加油压力） | 安全系数3.0 |

**关键设计挑战**：
- **快速对接密封**：Boom插入到密封建立<2秒，O形圈采用氟硅橡胶(FVMQ)耐Jet A的快速溶胀补偿
- **应急脱离**：信号触发→完全密封关闭<500ms，防燃油喷溅
- **冰堵塞**：高空加油（>25000ft），水汽凝结在受油阀密封面结冰。措施：电加热元件或引气加热阀体

### 64.3.2 软管-锥套（Probe-and-Drogue）加油阀

**系统概述**：美国海军/海军陆战队及欧洲标准。受油机头部的**受油探头（Probe）**插入加油机拖曳的锥套（Drogue）。受油阀集成在探头内部。

**阀门设计差异**：

软管-锥套系统加油流量较低（300-500 USGPM），但对密封要求更严：
- 探头对接存在角度偏差（最大±15°），阀门须有自动对中补偿
- 燃油流动压力辅助密封——加油压力越高密封越紧（自紧设计）
- 止回阀防止受油机侧燃油倒流（加油机压力更高）

### 64.3.3 空中加油阀门密封与快速对接

空中加油阀门的密封技术是核心技术壁垒之一：

- **材料选择**：FKM（氟橡胶）在Jet A中体积溶胀>10%，不可用；FVMQ（氟硅橡胶）溶胀<5%，耐低温至-60°C，为首选。新一代全氟醚橡胶（FFKM）耐温-40°C至+325°C、耐全化学介质，但成本极高（>$100/O-ring）。
- **自紧密封（Self-Energizing Seal）**：U形杯密封或V形组合密封，流体压力使密封唇更紧贴密封面，压差越大密封越好。
- **表面处理**：阀座锥面热处理硬度HRC55+，表面粗糙度Ra<0.2μm，镀铬或化学镀镍(Ni-P)防腐蚀。

## 64.4 燃油箱惰化阀

### 64.4.1 氮气惰化系统（NGS）阀门

TWA 800航班（1996年B747中央油箱爆炸）事故后，FAA发布SFAR 88，要求新机型必须配备燃油箱惰化系统（FTIS），降低油箱可燃蒸气浓度。

**NGS工作原理**：从发动机引气中分离氮气（通过空气分离模块ASM——中空纤维膜），产生富氮空气（NEA，氧含量<12%），充入燃油箱上部空间，将氧浓度降低到可燃极限以下（<9%O₂海平面，<12%O₂高空）。

**NGS阀门系统**：

| 阀门 | 功能 | 设计特征 |
|------|------|---------|
| 引气关断阀(BASOV) | 控制进入ASM的发动机引气 | 电控气动或电磁，fail-safe关闭 |
| 富氮空气分配阀(NDV) | 将NEA分配到各油箱 | 三通/四通旋转阀或多联电磁阀组 |
| 油箱隔离阀(TIV) | 防止油箱间气体流通 | 在加油/放油时关闭，维持惰化 |
| 排气阀/Vent Valve | 控制油箱压力 | 与大气连通，保持油箱内外压差<±2psi |
| 氧传感器 | 在线监测NEA氧含量 | 氧化锆传感器，反馈控制BASOV |

### 64.4.2 富氮空气分配阀

富氮空气分配阀（Nitrogen Distribution Valve, NDV）将ASM产生的NEA按需分配到各个燃油箱：

- 旋转盘阀设计：步进电机驱动带通道的陶瓷盘，通过角度控制分配比例
- 每个出口含流量限制孔（Flow Restrictor），平衡各管路阻力
- 失效安全位：断电停在中立位，各油箱等比例供气

### 64.4.3 氧浓度传感器与反馈控制

闭环惰化系统依靠氧传感器调节BASOV开度和ASM工作点：
- **氧化锆氧传感器**：安装在NEA总管上，量程0-25%O₂，精度±0.5%O₂
- **反馈控制逻辑**：O₂%>12%→增加BASOV开度→增加引气量→提高ASM氮气产量→降低NEA氧浓度
- **传感器防护**：阀前安装微粒过滤器（0.3μm）和除水器（膜式干燥器），防止传感器污染误报

## 64.5 应急放油阀

### 64.5.1 应急放油系统（Jettison）阀门

**功能需求**：大型运输类飞机最大起飞重量显著高于最大着陆重量（典型差值：B777约50,000kg），起飞后紧急返航时须快速放油减重至安全着陆重量。

适航条款FAR/CS/CCAR §25.1001规定：应急放油系统必须能将飞机重量在指定时间内降至安全着陆重量。

**燃油放油阀（Jettison Valve）**：

| 参数 | 典型要求 |
|------|---------|
| 放油速率 | 每分钟至少放油总燃油量的1%（按FAR 25.1001） |
| 阀门作动 | 电控液压作动或电动，失电关闭(fail-close) |
| 阀门位置 | 安装在翼尖或机翼后缘外侧，放油远离机身和襟翼 |
| 防误操作 | 双开关激活逻辑（两个独立开关同时操作） |
| 飞行限制 | 限制在襟翼收上时放油，防止燃油接触高温襟翼作动器 |

**阀门流通能力计算**：

放油阀流量由Torricelli方程修正：

$$Q = C_d A \sqrt{2g \cdot \Delta h + \frac{2\Delta P}{\rho}}$$

其中：Q为体积流量，C_d为流量系数(~0.65-0.8)，A为阀口面积，Δh为液面到阀口高度差，ΔP为油箱内压力差（含NGS增压）。

### 64.5.2 放油阀安全设计

- **防冰措施**：阀体和放油管内置电加热毯，防高空结冰堵塞
- **防静电**：阀体和管路全程电搭接（Bonding），电阻<1Ω，防静电火花引燃
- **防火**：放油管出口远离发动机和APU排气，通过防火区域时的管路防护套（Fire Sleeve）

## 64.6 燃油系统防火安全

### 64.6.1 燃油关断阀（Fire Shutoff Valve）

发动机防火关断阀（Engine Fire Shutoff Valve / Fuel Spar Valve）是燃油系统防火安全的最后一道防线。安装在**防火墙贯穿点（Firewall Penetration）**——通常是机翼前梁（Front Spar）位置。

**适航条款**：
- **FAR/CS/CCAR §25.1181**：指定防火墙的指定火区范围和贯穿规则
- **FAR/CS/CCAR §25.1183**：导管、管线和接头贯穿防火墙的要求
- **FAR/CS/CCAR §25.1189**：关断阀要求——至少一个燃油关断阀必须位于防火墙外（发动机侧）

**防火关断阀设计**：

| 特征 | 要求 |
|------|------|
| 安装位置 | 防火墙发动机侧，距防火墙≤6英寸 |
| 作动时间 | 全关<3秒（机组拉动T手柄/按压防火按钮后） |
| 防火能力 | 1100°C/15分钟（按ISO 2685/AC 20-135） |
| 材料 | 阀体：不锈钢17-4PH或Inconel 625；密封：石墨或柔性石墨 |
| 作动方式 | 电动马达驱动 + 防火手柄通过钢索机械直接作动（双模式冗余） |
| 失效安全 | 电信号失效：机械钢索仍可关闭；钢索断裂：电动仍可关闭 |

### 64.6.2 防火墙贯穿阀门密封

防火墙贯穿点的密封设计是防火认证中最严格的挑战：
- **Firewall Seal**：不锈钢波纹管密封或高温膨胀密封圈（石墨/蛭石复合材料），1100°C下保持结构完整性和气密性
- **导管与防火墙间隙**：≤3mm（FAR 25.1191），填充防火密封胶（如3M Fire Barrier Sealant CP 25WB+，1100°C认证）
- **热传导阻断**：导管通过防火墙段加装隔热套管（Insulating Sleeve）和散热法兰（Heat Sink Flange），防止阀门执行器过热

### 64.6.3 综合防火安全设计

燃油系统阀门防火安全设计遵循**多层防御（Defense-in-Depth）**原则：

1. **第一层：防漏设计**——阀门密封冗余，泄漏监测（油箱间燃油量异常变化检测）
2. **第二层：通风与惰化**——排放通风管路防止可燃油气积聚，NGS降低气相氧浓度
3. **第三层：隔离**——防火关断阀物理切断燃油供应，防火墙隔离火区
4. **第四层：扑灭**——灭火瓶手动释放，覆盖发动机和APU火区

## 本章小结

飞机燃油系统阀门覆盖了从油箱到发动机的完整供油链路，同时满足增压输送、交输备份、空中加油、惰化安全和应急放油等多维需求。燃油阀门设计在密封可靠性、防火安全性、全包线适应性三个维度上面临严峻挑战。波音787的全电燃油系统和F-35的PTMS一体化燃油热管理代表了两个不同的技术前沿方向，其阀门设计理念的差异值得深入对比学习。
