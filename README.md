## 系统定位

Avis 系统不只是手册，它是一个**可记录版本演化、支持共创的航空航天阀门研发智库平台**。系统以 INCOSE 系统工程方法论为设计哲学，将全套航空航天阀门研发设计手册（V5.0，约60万汉字，5大编57章）构建为可演化、可协作的知识基础设施。

核心价值主张：
- **版本演化**：从 V0.1（2025年）到 V5.0（2026年6月），完整记录知识迭代轨迹
- **开放共创**：RFC 式提案机制，支持领域专家协同贡献
- **知识图谱**：阀门分类学、材料体系、故障模式等本体化组织
- **工程严谨**：基于 INCOSE SE 方法论，五级引证金字塔，全量交叉引用

## 系统架构概览

Avis 系统采用四层架构：

| 层级 | 目录 | 职责 |
|------|------|------|
| **知识层** | `core/` | 手册本体，V5.0快照 + 历史版本归档 |
| **图谱层** | `knowledge-graph/` | 本体定义、术语词典、交叉引用索引 |
| **协作层** | `collaboration/` | 贡献指南、RFC提案、评审记录 |
| **工具层** | `tools/` | 自动化脚本、质量检查、构建配置 |
| **发布层** | `web/` | Web发布（预留） |
