# 附录A 航空航天阀门设计校验清单（Design Verification Checklist, DVC）

**作者**：小贤（航空航天阀门研发系统工程师，国际系统工程师CSEP，战略科学家）
**版本**： V5.0 | **类型**：工具附录 | **适用阶段**：PDR后/CDR前设计审查

---

## A.0 使用说明

本设计校验清单（DVC）是手册各章设计约束的系统性汇总，用于在初步设计评审（PDR）之后、关键设计评审（CDR）之前，对阀门设计进行逐项自检。每条校验项分为三个等级：

- **M（Must，强制性）**：违反将导致设计不合格——安全/任务关键约束，无任何豁免可能。
- **S（Should，应满足）**：违反需提交书面偏离申请（Deviation Request）并经总师审批——设计优化目标，有明确的工程代价。
- **R（Recommended，推荐）**：违反无需正式偏离申请但应在设计评审中说明理由——工程最佳实践。

校验结果标记：
- ✅ 通过
- ❌ 不通过（需整改）
- ⚠️ 条件通过（需额外分析或试验验证）
- N/A 不适用

设计师应在CDR评审前逐项完成本清单并签字，纳入设计档案。

### 表A.1-1：阀门安全裕度标准

| 失效模式 | 最低安全裕度（鉴定件） | 最低安全裕度（验收件） | 分析方法 |
|---------|---------------------|---------------------|---------|
| 承压壳体爆破/破裂 | ≥1.50（极限压力） | ≥1.25（极限压力） | FEA + 液压爆破试验 |
| 承压壳体屈服 | ≥1.25（屈服压力） | ≥1.10（屈服压力） | FEA + 应变片验证 |
| 紧固件断裂 | ≥1.40（极限载荷） | ≥1.15（极限载荷） | 手算/FEA + 接头试验 |
| 弹簧疲劳断裂 | ≥1.33（无限寿命设计）或 ≥1.1×（任务循环次数） | 同鉴定 | Goodman/Soderberg疲劳分析 + 疲劳试验 |
| 密封面泄漏（结构变形导致） | FEA变形量≤密封件补偿能力的50% | 同鉴定 | FEA + 氦检漏试验 |
| 焊缝断裂 | ≥1.40（极限载荷，计入焊趾应力集中系数） | ≥1.15（极限载荷） | FEA + 焊缝试样拉伸 |
| 电磁力不足（阀门无法开启） | 最小电压（含母线压降）×0.8下的电磁力 ≥ 阀门开启所需力的1.5倍 | 同鉴定 | 电磁FEA + 动作试验 |
